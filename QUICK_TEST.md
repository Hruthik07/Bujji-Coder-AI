# Quick Test Instructions

## ⚡ 30-Second Test

1. **Check Backend:**
   ```bash
   curl http://localhost:8001/api/status
   ```
   Should return JSON with `assistant_ready: true`

2. **Open Browser:**
   - Go to: `http://localhost:3001`
   - Should see the AI Coding Assistant UI

3. **Test Chat:**
   - Type: "Hello, what can you do?"
   - Should receive a response
